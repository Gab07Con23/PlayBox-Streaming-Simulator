PlayBox is a console‑based simulation of a streaming platform built in C#.
The project models real-world streaming features using object‑oriented programming concepts:
- Abstract base class for shared content
- Inheritance for Movies, Series, and Episodes
- User profiles with watchlists and history
- Subscription plans
- Playback simulation (play, pause, stop, resume)
- Recommendation engine
- Clean class structure and UML design
This repository contains the full source code, UML diagram, and documentation for the capstone project.
